=============

A Bucklescript binding of svgjs 2.6.

Setup
------

```bash
npm install
```

Build
------

```bash
./build.sh
```

Run
---

Run http server

```bash
./run.sh
```
browse http://localhost:8080/examples/index.html

Build doc on Mac
--------------------

```bash
yarn add bsdoc@6.0.4-alpha

./build.sh
npm run doc-init
npm run build:doc
```

[./docs](./docs) coantains document of svgjs
