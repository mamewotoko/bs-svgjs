=============

A Bucklescript binding of svgjs 2.6.

Setup
------

```
npm install
```

Build
------

```
sh build.sh
```

Run
---

browse index.html 

```
python3 -m http.server
```
browse http://localhost:8000


Build doc
------------

```
./build.sh
npm run doc-init
npm run doc
```

[./docs](./docs) coantains document of svgjs

-----------
