bs-svgjs
=============

A Bucklescript binding of svgjs 2.6.

Setup
------

```
npm install
```

Build
------

```
sh build.sh
```

Run
---

browse index.html 

```
python3 -m http.server
```
browse http://localhost:8000

Reference
-----------
* [SVG.js](https://svgjs.com/docs/3.0/)
